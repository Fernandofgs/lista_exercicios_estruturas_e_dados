4) Verifique o programa abaixo. Encontre o seu erro e corrija-o para que escreva o
número 10 na tela (mantenha a variável q como parâmetro do printf).
#include <stdio.h>
int main()
{
int x, *p, **q;
p = &x;
q = &p;
x = 10;
printf("\n%d\n", &q);
return(0);
}
basta mudar 
printf("\n%d\n", &q); 
para 
printf("\n%d\n", **q); 
 para que se imprima o valor 10
